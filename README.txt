hariraj-portfolio (Vite + React + Tailwind)

How to use:
1. Extract this folder.
2. Replace /public/assets/profile-placeholder.jpg with your profile image named profile.jpg OR keep placeholder.
3. Replace /public/assets/resume-placeholder.docx with your resume and rename to resume.docx
4. In project root run:
   npm install
   npm run dev
5. Open http://localhost:5173

Notes:
- Resume download link points to /assets/resume.docx (place your file there).
- Profile image path: /public/assets/profile.jpg. If not present, placeholder will show.
